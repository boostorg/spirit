Expression

Attribute

Description

stream

hold_any

Generates narrow character (char) based output using the matching streaming operator<<()

stream(s)

Unused

Generates narrow character (char) based output from the immediate argument s using the matching streaming operator<<()

wstream

hold_any

Generates narrow character (wchar_t) based output using the matching streaming operator<<()

wstream(s)

Unused

Generates wide character (wchar_t) based output from the immediate argument s using the matching streaming operator<<()
